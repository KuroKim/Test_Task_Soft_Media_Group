# Микросервис для сокращения ссылок

Бэкенд-сервис на FastAPI для сокращения URL-адресов.

## ✨ Функционал

-   **Сокращение URL**: Эндпоинт `POST /shorten` для создания короткого идентификатора.
-   **Редирект**: Эндпоинт `GET /{short_id}` для перенаправления на оригинальный URL.
-   **Счетчик переходов**: Атомарное увеличение счетчика кликов при каждом редиректе.
-   **Статистика**: Эндпоинт `GET /stats/{short_id}` для получения статистики по ссылке.

## 🛠 Технологический стек

-   **Язык**: Python 3.12
-   **Фреймворк**: FastAPI
-   **База данных**: PostgreSQL
-   **ORM**: SQLAlchemy (Async)
-   **Миграции**: Alembic
-   **Тестирование**: Pytest, httpx
-   **Контейнеризация**: Docker

## 🚀 Быстрый запуск

Проект полностью контейнеризирован и запускается одной командой.

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/KuroKim/Test_Task_Soft_Media_Group
    cd Test_Task_Soft_Media_Group
    ```

2.  **Запустите сервисы с помощью Docker Compose:**
    ```bash
    docker-compose up --build
    ```
    *Эта команда соберет образ, дождется готовности базы данных, автоматически применит все миграции (Alembic) и запустит веб-сервер.*

3.  **API готово к работе:**
    -   **Интерактивная документация (Swagger UI)**: [http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs)

## ✅ Запуск тестов

В проекте настроена отдельная база данных для проведения интеграционных тестов.

1.  **Запустите тестовую базу данных:**
    ```bash
    docker-compose up -d test_db
    ```

2.  **Запустите тесты:**
    ```bash
    pytest
    ```
    *Эта команда выполнит все тесты из папки `tests/`.*
